//-----2022.6.8---------//
网站的静态页面结构基本完成，在整体布局改动不大情况下后续对其功能进行简洁化，对表现形式进行修改。
功能：
    - 将网站的头部的header nav和尾部的footer, 统一放在了base文件夹下。这样在任何页面通过jquery的load方法就可以调用该头尾部，以方便修改后全部页面都适用。
    - Academic Communication卡片页面，通过input和translate,实现学术卡片的选择和切换，transition实现平滑过渡效果。点击卡片实现跳转。
    - People页面以卡片形式提供每一个信息。侧边栏可以选择跳转分组。
    - Publication 页面也采用卡片提供信息，DOI 和arXiv按钮可以跳转到论文期刊(下载)页面。侧边栏可以跳转分组。
    
行为：
    - 网页对css文件夹下文件以及base文件夹下文件的引用统一采用从根目录 / 开始。
    - img目录一般都是与它同级下html引用图片的集合目录。
    - 将为news, sparetime, academic目录下创建articles目录用于存放该栏目的所有文章。并在articles目录下单独设置文字的css样式。
